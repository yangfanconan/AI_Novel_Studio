# AI Novel Studio v0.2.0 发布说明

## 🎉 新功能

### AI 辅助创作
- **AI 生成角色** - 根据项目背景自动生成角色信息（姓名、外貌、性格、背景）
- **AI 生成角色关系** - 智能分析并建议角色之间的关系网络
- **AI 生成世界观** - 根据项目类型生成世界观设定
- **AI 生成情节点** - 根据剧情走向自动生成情节点建议
- **AI 一键排版** - 智能格式化文本，对话内容自动加粗

### 多媒体生成
- **分镜脚本生成** - 根据剧情内容生成结构化分镜描述
- **漫画分镜** - 自动将文本转换为漫画分镜
- **插画生成** - 根据描述生成场景/角色插画
- **剧本转换** - 小说文本转换为标准剧本格式

### 插件系统
- **插件管理器** - 完整的插件安装、启用/禁用、卸载功能
- **插件开发指南** - 内置插件开发文档和系统 API 说明
- **示例插件** - 字数统计工具、灵感生成器示例

### 版本控制
- **项目快照** - 创建项目历史快照
- **版本比较** - 对比不同版本的差异
- **回滚功能** - 恢复到历史版本

### 角色对话
- **AI 角色模拟** - 与角色进行对话，深化角色塑造
- **系统提示词** - 自定义角色对话的 AI 行为

### 角色成长轨迹
- **成长记录** - 记录角色在不同章节的变化
- **时间线展示** - 可视化角色发展历程

### 写作工具
- **敏感词检测** - 检测并标记敏感词汇
- **错别字检查** - 检查常见错别字
- **语法检查** - 基础语法检查
- **格式规范化** - 文本格式统一

### 文本分析
- **写作风格分析** - 分析写作风格特点
- **节奏分析** - 情节节奏可视化
- **情感分析** - 文本情感走向
- **可读性评估** - 自动评估文章可读性

### 导出功能
- **多格式导出** - 支持 PDF、EPUB、TXT、Markdown、DOCX 格式
- **项目/章节导出** - 可导出整个项目或单个章节

### 知识库
- **知识条目管理** - 创建、编辑、搜索知识条目
- **关联关系** - 建立知识条目之间的关联

### 阅读模式
- **沉浸式阅读** - 舒适的章节阅读体验
- **个性化设置** - 字体大小、背景色可调节

## 🐛 Bug 修复
- 修复项目删除功能不工作的问题
- 修复章节删除功能不工作的问题
- 修复对话框不显示的问题
- 修复插件管理器白屏问题
- 修复章节保存重复创建的问题

## 🔧 系统改进
- **模型设置增强** - 支持默认模型选择、AI 参数配置
- **API 密钥管理** - 安全存储和管理 API 密钥
- **错误边界** - 添加全局错误处理，提升稳定性

## 📦 技术栈
- 前端: React + TypeScript + Tailwind CSS
- 后端: Rust + Tauri
- 数据库: SQLite
- AI: 支持智谱 GLM、OpenAI、Ollama 等多模型
