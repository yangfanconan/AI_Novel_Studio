# AI小说创作工作室 - 项目概览

## 项目简介

**AI小说创作工作室** 是一款专业级AI小说创作工具，具备傻瓜式操作体验。核心差异化功能是从小说文本自动生成专业的分镜脚本、漫画、插画、动画等多媒体内容。

---

## 核心特性

### 🎯 产品定位
- **功能专业**：提供专业作家级别的创作工具
- **使用简单**：傻瓜式操作，降低学习门槛
- **本地优先**：支持完全离线使用，保护创作隐私

### ⭐ 差异化功能
1. **分镜脚本生成**：从小说自动生成专业影视分镜
2. **漫画生成**：智能分格、AI绘图、角色一致性
3. **插画生成**：场景插画、角色立绘、封面设计
4. **动画生成**：关键帧动画、镜头运动

### 🔧 技术特点
- **多模型支持**：OpenAI、Claude、Ollama、文心、通义等
- **插件系统**：灵活扩展，无限可能
- **脚本支持**：Python/JavaScript自动化
- **跨平台**：Windows、macOS、Linux

## 技术栈概览

### 前端
- **框架**: Tauri 2.0 + React 18 + TypeScript 5
- **UI**: TailwindCSS + Shadcn UI
- **编辑器**: Slate.js (富文本) + CodeMirror (代码)
- **可视化**: D3.js + React Flow

### 后端
- **核心**: Rust (Tauri后端)
- **AI服务**: Python 3.11+ + LangChain
- **数据库**: SQLite + SQLCipher加密
- **图像生成**: ComfyUI / Stable Diffusion

### AI模型
- **云端**: OpenAI、Claude、Gemini、文心、通义、智谱GLM
- **本地**: Ollama、vLLM、LocalAI

## 快速开始

### 环境要求
- Node.js 18+
- Rust 1.70+
- Python 3.11+ (可选)

### 安装依赖
```bash
# 前端依赖
cd frontend
npm install

# 后端依赖 (Rust自动处理)
cd ../src-tauri
cargo build
```

### 启动开发服务器
```bash
npm run tauri dev
```
