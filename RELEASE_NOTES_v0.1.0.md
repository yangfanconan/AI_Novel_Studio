# AI Novel Studio v0.1.0

## 🎉 首个正式版本发布！

AI Novel Studio 是一款现代化、智能化的小说创作软件，结合人工智能技术，帮助作家更高效地进行文学创作。

---

## ✨ 核心功能

### 📝 项目与章节管理
- 多项目管理，轻松切换不同创作任务
- 章节创建、编辑、删除、排序
- 实时字数统计
- 自动保存功能

### 👤 角色档案系统
- **基本信息**：姓名、身份、种族、性别、年龄等
- **性格分析**：MBTI、九型人格、八字命理、紫微斗数
- **能力装备**：技能列表、持有物品
- **事件时间线**：记录角色经历的重要事件
- **角色头像**：支持上传角色形象

### 🌍 世界观设定系统
- 分类管理世界观元素（地理、历史、文化、魔法体系等）
- 标签系统便于检索
- 世界观事件时间线
- 支持AI生成世界观内容

### 📚 知识库管理
- 集中管理所有创作素材
- 一键同步角色和世界观数据
- 支持关键词搜索
- 条目关联与引用

### 🤖 AI 写作助手
- **智能续写**：根据上下文自动生成后续内容
- **内容优化**：润色文字，提升文采
- **续写选项**：提供多个不同方向的续写建议
- **一致性检查**：检测角色设定、世界观冲突

### 🔗 角色关系图谱
- 可视化展示角色关系
- 支持多种关系类型（亲情、友情、爱情、敌对等）
- AI 辅助生成角色关系

### 🌳 情节树管理
- 树状结构组织故事情节
- 支持多级情节嵌套
- 拖拽排序
- 情节状态追踪

### 🔌 多模型支持
- 智谱 GLM 系列（glm-4, glm-4-plus, glm-4-flash 等）
- OpenAI 模型（需配置 API Key）
- Ollama 本地模型
- 自定义模型接入

| 层级 | 技术 |

## 📦 安装说明

### 系统要求
- macOS 10.15+
- Windows 10+
- Linux (主要发行版)

### 安装方式
1. 下载对应平台的安装包
2. 安装并启动应用
3. 在设置中配置 AI 模型 API Key（如使用智谱 GLM，需申请 API Key）

---

## 🔑 AI 模型配置

### 智谱 GLM（推荐）
1. 访问 [智谱开放平台](https://open.bigmodel.cn/) 注册账号
2. 获取 API Key
3. 在应用设置中填入 API Key

### OpenAI
1. 获取 OpenAI API Key
2. 在应用设置中配置

### Ollama 本地模型
1. 安装 Ollama
2. 下载所需模型
3. 在应用中配置 Ollama 地址
